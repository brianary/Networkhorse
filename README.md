Useful https and network utilities.

<img src="images/demo.gif" alt="a demonstration of the module commands" height="300" width="600" />

- [Connect-SshKey](https://github.com/brianary/Networkhorse/wiki/Connect-SshKey): Uses OpenSSH to generate a key and connect it to an ssh server.
- [ConvertTo-BasicAuthentication](https://github.com/brianary/Networkhorse/wiki/ConvertTo-BasicAuthentication): Produces a basic authentication header string from a credential.
- [ConvertTo-MultipartFormData](https://github.com/brianary/Networkhorse/wiki/ConvertTo-MultipartFormData): Creates multipart/form-data to send as a request body.
- [Get-ContentSecurityPolicy](https://github.com/brianary/Networkhorse/wiki/Get-ContentSecurityPolicy): Returns the content security policy at from the given URL.
- [Get-Dns](https://github.com/brianary/Networkhorse/wiki/Get-Dns): Looks up DNS info, given a hostname or address.
- [Get-ServerCertificate](https://github.com/brianary/Networkhorse/wiki/Get-ServerCertificate): Returns the certificate provided by the requested server.
- [Get-SslDetails](https://github.com/brianary/Networkhorse/wiki/Get-SslDetails): Enumerates the SSL protocols that the client is able to successfully use to connect to a server.
- [Save-PodcastEpisodes](https://github.com/brianary/Networkhorse/wiki/Save-PodcastEpisodes): Downloads enclosures from a podcast feed.
- [Save-WebRequest](https://github.com/brianary/Networkhorse/wiki/Save-WebRequest): Downloads a given URL to a file, automatically determining the filename.
- [Show-HttpStatus](https://github.com/brianary/Networkhorse/wiki/Show-HttpStatus): Displays the HTTP status code info.
- [Test-HttpSecurity](https://github.com/brianary/Networkhorse/wiki/Test-HttpSecurity): Scan sites using Mozilla's Observatory.
- [Trace-WebRequest](https://github.com/brianary/Networkhorse/wiki/Trace-WebRequest): Provides details about a retrieving a URI.
